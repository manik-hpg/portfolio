## Overview

Co-founded Solvfit, a skill-based talent matching platform designed to reduce hiring biases and improve placement precision through transparent, explainable algorithms.

## Key Achievements

- **Improved placement precision by 40%** through skill-based matching algorithms
- **Formed early industry partnerships** with companies looking for unbiased hiring solutions
- **Led product development** from ideation to launch

## Technical Implementation

### Frontend
- React with TypeScript for type-safe development
- Tailwind CSS for rapid, consistent styling
- Responsive design for all device sizes

### Backend & Infrastructure
- Node.js API server
- AWS deployment (EC2, S3)
- Scalable architecture for growth

## Product Features

- Skill assessment and scoring system
- Explainable matching algorithms (candidates can see why they matched)
- Company dashboards for managing candidates
- Candidate profiles with verified skill badges
